# 数据可视化Demo

## 本软件是一款基于 Python + Flask + PyWebView + Matplotlib 开发的桌面数据可视化分析工具
系统支持多种图表类型的数据分析与可视化生成，并提供统计指标计算功能，目前能够实现：

<li>折线图
<li>柱状图
<li>直方图
<li>散点图(未完成)
<li>饼图（未完成）
<li>箱线图（未完成）

###### 作者非计算机科班专业，本软件作为作者的软件前后端练手项目，未来将会不定期更新，并尝试加入一些新功能。

  ------
  
  ## 使用方式
  下载本仓库后运行 main.py即可进如前端软件桌面，跟随指示输入数据即可
